To-Do List CLI App (Python)

A simple command-line To-Do app built with Python.

Features

- Add tasks
- View tasks
- Delete tasks
- Mark tasks as completed ✔
- Persistent storage (tasks saved to file)
- Date & time tracking

Tech

- Python (standard library)

How to Run

1. Make sure Python is installed
2. In terminal:
   python todo.py

File Structure

- todo.py → main program
- tasks.txt → saved tasks (auto-created)
